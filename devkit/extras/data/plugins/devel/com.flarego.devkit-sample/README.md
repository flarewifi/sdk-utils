# Devkit Sample

A minimal starter plugin for Flarewifi — adds a single working admin page you
can build on when creating your own plugin for a hotspot business.

> This file is a template. When you copy this plugin as your starting point,
> replace everything below with your own plugin's pitch, written the way a
> real plugin README should be:

## How to write your plugin's README

Once published, this file becomes your plugin's **store listing description**
— the copy an operator reads while deciding whether to buy/install it. Write
it for that operator, not for another developer:

- **Open with a short pitch** (2-4 sentences) — what problem it solves for
  their hotspot business, in plain language.
- **List benefits, not implementation** — 4-8 bullets on what the operator or
  their customers get, not how it's built.
- **Mention real prerequisites only if a buyer needs to know before
  purchasing** (e.g. specific hardware required), in plain language.
- **Leave out**: "How It Works" internals, config field tables, API/SDK
  references, file paths, and license/package metadata.
