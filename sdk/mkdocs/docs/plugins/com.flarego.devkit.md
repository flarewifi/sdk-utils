# Devkit

The Devkit is the plugin that powers your Flarewifi development experience — the
admin/portal theme and the developer panel you use to upload, clone, and install
plugins while building them. It ships with the SDK and is what most of these
tutorials assume you're running.

## Open source — and open to contributions

The Devkit is a **free, open-source plugin**, and we'd love your help making it
better. If a workflow feels clunky, a feature is missing, or you spot a bug while
building your own plugin, please improve it and send the change back — everyone
building on Flarewifi benefits.

### How to contribute

1. Fork and clone the repository:

   ```sh
   git clone https://github.com/flarewifi/com.flarego.devkit
   ```

2. Load it into your devkit's `data/plugins/devel/` directory and make your change —
   whether it's a new developer-panel feature, a UI fix, or better error messages.
3. Test it against a running SDK, commit on a feature branch, and
   **open a pull request** describing what you improved and why.

Small, focused PRs are the easiest to review and merge. Bug reports and feature
ideas are welcome too — open an issue if you're not ready to send code yet.
